# Funciones(Metodos) en Java

1. Que es una Funcion en Java?

En java, una funcion se llama `metodo`

un metodo es un bloque de codigo:

- Realiza una Tarea especifica
- Puede recibir datos (paramaetros)
- Puede devolver un resultado
- Puede Reutilizarse

2. Estructua de un metodo

```Java
modificador static tipoRetorno nombreMetododo(parametros){
    //Cuerpo
    return valor; // Aplica
}
```

3. Por que usamos metodos estaticos?

por que el metodo main es **Estatico**

Un metodo Estatico: 

- Pertenece a la clase
- No se necesita crear un objeto
- Se invoca directamente
